Footprint

Introduced 0.2.0 AutoCAD Civil 3D 2019+ Update in 0.4.0

Overview

  • Use the footprint command to quickly draft perimeters of features such as houses.
  • Select a start point ⇒ select an angle ⇒ Enter a distance (repeating) ⇒ Exit command
  • Distance entry methods:
    • 5.00 - Create a line 5.00 feet in length 90째 clockwise from current angle.
    • -5.00 - Create a line 5.00 feet in length 90째 counter-clockwise from current angle.
    • 5.00@45 - Create a line 5.00 feet in length 45째 clockwise from current angle.
  • Once the command is exited, a closure report will be generated and outputted in the command area.

Description

The footprint command allows the drafter to select a starting point in a drawing, establish a starting bearing, and quickly input distance at default 90째 angles (or at an angle you specify). Positive distance will move clockwise while negative distances will move counter-clockwise. When you are completed with your footprint, you can press enter or escape to complete the command, and a closure report will be outputted to the command area. The closure report will state the overall misclosure, the North-South component, and the East-West component. 0.4.0 Also included in the closure report is the area of the footprint in square feet and acres.

The command does assume the elevation of your starting position. If you have OSNAPZ set to 0 and selected a point with an elevation (i.e.100.00'), all the lines created in the command will be created at elevation 100.00'. This is to ensure that the lines can be joined into a 2D polyline if needed afterwards.
